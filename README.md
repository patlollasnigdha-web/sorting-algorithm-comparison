Sorting Algorithm Comparison

=>Project Overview
This project is a web-based application that compares the performance of sorting algorithms such as Bubble Sort and Merge Sort.
It allows users to input custom data and measures the execution time using Python. 
The results are visually displayed using graphs for easy comparison. 
The system is built using Flask for the backend and HTML, CSS, and JavaScript for the frontend. 
It helps in understanding algorithm efficiency in a simple and interactive way.

=>Features
*  User input for custom number sequences
*  Time calculation using Python
*  Graph visualization of sorting performance
*  Simple frontend using HTML, CSS, and JavaScript
*  Backend powered by Flask

=>Technologies Used
* Python (Flask)
* HTML, CSS, JavaScript
* Chart.js (for graph visualization)
